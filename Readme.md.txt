
## How to Use 🚀

### Option 1: Quick Start
1. Download all 3 files into the same folder
2. Open `index.html` in your browser
3. Start tracking your water consumption!

### Option 2: Live Server (Recommended)
1. Install Visual Studio Code
2. Install "Live Server" extension
3. Right-click `index.html` → "Open with Live Server"
4. The app will open at `http://localhost:5500`

## Troubleshooting 🔧

### Problem: Data not saving
**Solution:** Check browser settings allow localStorage

### Problem: Charts not loading
**Solution:** Ensure internet connection for Chart.js CDN

### Problem: Not working on mobile
**Solution:** Clear browser cache and reload

### Problem: Slow performance
**Solution:** Close other tabs, clear browser cache
